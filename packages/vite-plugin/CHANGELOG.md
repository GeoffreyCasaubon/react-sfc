# @g-casau/rsfc-vite-plugin

## 1.0.0

### Major Changes

- Initial stable release — 1.0.0.

  Full feature set:

  - `@g-casau/rsfc-core` — zero-dep parser + generator, `defineProps<T>()` macro (incl. multi-line types), scoped/module CSS, CSS preprocessors (Sass/Less/Stylus), V3 source maps
  - `@g-casau/rsfc-vite-plugin` — Vite 5+ integration, smart HMR (style-only vs full), custom block transforms
  - `@g-casau/rsfc-webpack-loader` — Webpack 5+ / Next.js 15 integration
  - `@g-casau/rsfc-cli` — `rsfc compile` (standalone JS via esbuild) + `rsfc parse`
  - `@g-casau/rsfc-typescript-plugin` — tsserver Language Service Plugin for prop inference in any IDE

### Patch Changes

- Updated dependencies
  - @g-casau/rsfc-core@1.0.0
